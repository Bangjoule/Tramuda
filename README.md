## Run Locally

Clone the project

```bash
  git clone https://github.com/ahmadUffi/Tramuda.git
```

Go to the project directory

```bash
  cd tramuda
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run start
```
